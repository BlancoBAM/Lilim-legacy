# Changelog

All notable changes to Lilith TTS will be documented in this file.

## [0.1.0] — 2026-03-10

### Added
- Pure-Rust TTS inference using NeuTTS Nano model
- CLI with `speak`, `download`, and `info` commands
- Voice cloning from audio samples (`.npy` reference codes)
- WAV file and in-memory byte output
- System tray panel with voice selection and speed control
- "Create a new voice" workflow (sample → encode → preview → save)
- Ctrl+Shift+T hotkey integration for read-aloud
- Pre-loaded American English voices
